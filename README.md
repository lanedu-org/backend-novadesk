# NovaDesk Solutions – Backend

Sistema backend para la gestión interna de clientes y tareas de NovaDesk Solutions.

Este proyecto forma parte del programa LANEDU y simula el trabajo real de un Backend Developer.

## Objetivo
Construir un sistema simple para:
- Gestionar clientes
- Gestionar tareas
- Asignar trabajo
- Llevar seguimiento de estados

## Documentación
Los requerimientos y reglas de negocio se encuentran documentados en Notion.

## Flujo de trabajo
- Las tareas se gestionan como Issues
- El avance se controla en GitHub Projects
- Cada Issue representa una tarea técnica concreta
